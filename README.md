# 学習記録アプリ

## アプリ概要

日々の学習内容を記録し、学習時間や継続状況を可視化する Web アプリです。

## 作成背景

React・TypeScript の基礎理解から、コンポーネント設計、状態管理、データ保存、GitHub 運用、デプロイまで一連の開発経験を積むための学習プロジェクトです。

## 使用技術

- React
- TypeScript
- Vite
- localStorage（データ保存）
- Git / GitHub
- Vercel（デプロイ予定）

## 機能一覧

- [ ] 学習記録の登録
- [ ] 学習記録の一覧表示
- [ ] 学習記録の削除
- [ ] 合計学習時間の表示（総・今日・今週）
- [ ] localStorage へのデータ保存
